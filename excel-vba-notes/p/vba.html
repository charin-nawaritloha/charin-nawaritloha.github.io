หน้าหลัก รับเขียน VBA วิธีเปิด/ปิด แมโคร แหล่งอ้างอิงข้อมูล ผู้เขียน

รับเขียน Macro VBA บน Excel

หากใครมีงานประมวลผลบน Excel ที่ซ้ำ ๆ ซาก ๆ หรือต้องใช้แรงงานคนเยอะ และต้องทำบ่อย ๆ หากสนใจอยากลองจ้างผมทำเป็นแมโคร ด้วย Visual Basic for Application (VBA) ได้ครับ

รายละเอียดงานที่รับ

  • รับทำบน Excel 2010 ขึ้นไป บนระบบปฏิบัติการ Windows เท่านั้น ไม่รับทำบน macOS
  • นำข้อมูลดิบมาจัดรูปแบบใหม่ตามที่ต้องการ เช่น Custom template, Table, PrivotTable
  • รับทำ Custom form เช่น งานตัวอย่าง บิล123
  • โปรแกรมและรหัสโปรแกรม (Source code) เป็นของคุณ 100%
  • ไม่รับเขียน .Net
  • ไม่รับงานจบ งาน project นิสิตนักศึกษา หรืองานการกุศล ครับ
เขียน VBA เพื่อจัดข้อมูล CSV ให้ได้รายงานตามรูปแบบที่ต้องการ (ใช้เวลา 1 - 2 วัน ตามความซับซ้อน)

ค่าบริการ

สำหรับค่าบริการ จะคิดตามจำนวนวันที่เขียนโปรแกรมครับ ส่วนเงื่อนไขก็เหมือนกับการรับงานทั่ว ๆ ไป

  • ใช้เวลาเขียนไม่เกิน 1 วัน คิด 3,000 บาท
  • วันที่ 2 - 3 คิดวันละ 2,000 บาท
  • ถ้าเกิน 3 วัน คิดวันละ 1,500 บาท
  • ไม่รับงานที่ต้องใช้เวลาเกิน 5 วันครับ 
  • มัดจำก่อนเริ่มงาน 50%
  • ในกรณีที่งานผิด แก้ไขงานได้ 2 ครั้ง
  • ถ้ามีแก้ไขงานเพิ่มเกินกว่าที่ตกลงไว้
    ถ้าคิดคำนวณแล้วใช้เวลาไม่เกิน 15% ของเวลารวม หยวน ๆ ทำแถมให้ฟรีครับ
  • 1 วัน = 8 ชม.
  • หากต้องการคู่มือ หรือ ต้องการให้ใส่ comment ใน source code อย่างละเอียด คิดค่าบริการอัตราเดียวกันกับที่กล่าวมาข้างต้น

ตัวอย่าง งานใช้เวลา 4 วัน จะคิดค่าแรง 3,000 + 2,000 + 2,000 + 1,500 = 8,500 บาท
