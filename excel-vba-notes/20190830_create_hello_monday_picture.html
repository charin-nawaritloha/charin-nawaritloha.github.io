หาฟอนต์เพื่อใช้ทำตัวหนังสือสวัสดี

หากในเครื่องยังไม่มีฟอนต์สำหรับสร้างตัวหนังสือเก๋ ๆ ล่ะก็ เดี๋ยวนี้เขามีทำแจกให้ใช้งานฟรี (อาจมีเงื่อนไขการใช้งานหากนำในการใช้ทางธุรกิจที่ทำให้เกิดรายได้) นั้นคือเว็บ f0nt.com มีให้เลือกมากมายจนตาลายกันเลย

ตัวอย่างฟอนต์ลายมือบางส่วน ดาวน์โหลดได้ที่f0nt.com

นอกจากนี้ยังมี 13 ฟอนต์ฟรีมาตรฐาน จากสำนักงานส่งเสริมอุตสาหกรรมซอฟต์แวร์แห่งชาติ (SIPA) ที่ควรมีไว้ติดเครื่องกัน

ฟอนต์ฟรีจาก SIPA ดาวน์โหลดได้ที่ f0nt.com

วิธีการติดตั้งฟอนต์ที่ดาวน์โหลดมา ก็ไม่ยาก ในเว็บ f0nt.com มีเขียนเอาไว้อยู่ ถ้าเป็น Windows 10 (จริง ๆ ก็เหมือน Windows 7 นั้นแหละ) ไม่ยาก ๆ ให้ทำตามลำดับดังนี้

  1. คลิกที่ลิงก์ดาวน์โหลดฟอนต์ที่ต้องการ
  2. บันทึกไฟล์ที่ดาวน์โหลด
  3. ให้ทำการ Unzip ไฟล์ที่ดาวน์โหลด จะได้ไฟล์ที่มีนามสกุลเป็น *.ttf
  4. จากนั้นให้ดับเบิ้ลคลิกที่ไฟล์ *.ttf เพื่อดูตัวอย่างแบบอักษร
  5. คลิกที่ปุ่ม Install เพื่อติดตั้ง
  6. เปิดโปรแกรม Excel มา ตรงที่เลือกแบบอักษร จะมีฟอนต์ที่เราติดตั้งเข้าไป สามารถใช้งานได้แล้ว

แต่งภาพใส่ตัวหนังสือ

การแทรกภาพ

ให้เตรียมชีทว่าง ๆ เปล่า ๆ มาไว้สำหรับนำภาพที่ดาวน์โหลดมาแปะ โดยการใช้คำสั่ง Insert → Picture จากนั้นให้คลิกเลือกเปิดไฟล์ที่ต้องการ

คำสั่งแทรกภาพ Insert → Picture
เลือกไฟล์ที่เตรียมไว้ คลิกปุ่ม Insert
ภาพที่เลือกจะถูกวางบนพื้นที่วางบนชีทที่กำลังทำงานอยู่

ย่อขยาย และการซ่อนบางส่วนของภาพ

ในกรณีที่ภาพมีขนาดใหญ่เกินไป สามารถใช้คำสั่งย่อขยาย ให้คลิกที่ภาพเพื่อเลือกภาพ จากนั้นย่อโดยการใช้เมาส์คลิกค้างตรงมุมภาพ จากนั้นเลื่อนย่อภาพให้เล็กลงตามต้องการ แล้วจึงปล่อยเมาส์

การย่อภาพ

หากภาพมีส่วนที่ไม่ต้องการ เช่น ภาพเป็นสี่เหลี่ยมผืนผ้า แต่ต้องการทำเป็นภาพจัตุรัส ก็สามารถใช้คำสั่ง Crop เพื่อซ่อนพื้นที่ที่ของภาพที่ไม่ต้องการได้ วิธีการคือ คลิกเลือกที่ภาพ จากนั้นเลือกเมนู Picture Tools → Format → Crop จากนั้นคลิกค้างที่แถบสีดำที่บริเวณขอบภาพ เลื่อนเมาส์จนได้ขนาดที่ต้องการจึงปล่อย นอกจากนี้ยังสามารถใช้ขนาดมาตรฐานที่โปรแกรมมีมาให้แล้วก็ได้

การซ่อนบางส่วนของภาพด้วยเมาส์
การซ่อนบางส่วนของภาพ ด้วยขนาดที่โปรแกรมกำหนดมาให้แล้วอัตโนมัติ

การใส่ตัวหนังสือ

หลังจากที่ทำภาพให้ได้ขนาดตามที่ต้องการแล้ว ขั้นต่อไปคือการใส่ตัวหนังสือเข้าไปในภาพ ขั้นตอนมีดังนี้

  1. เลือกเมนู Insert → Text Box
  2. คลิกบนภาพเพื่อสร้าง Text Box
  3. พิมพ์ข้อความที่ต้องการลงไป เช่น สวัสดีวันจันทร์
  4. หากต้องการเปลี่ยนฟอนต์ สามารถเลือกได้จากเมนู Home → Font เช่น เลือกฟอนต์ชื่อ Little_Star
  5. ปรับขนาดจากเมนู Home → Font Size
  6. ทำการใส่รูปแบบกราฟฟิกตัวอักษร จากเมนู Drawing Tools → Format → WordArt Style
  7. ปรับ Effect อื่น ๆ ได้ตามต้องการที่เมนู Drawing Tools → Format → Format Text Effects
ขั้นตอนการใส่ตัวหนังสือเข้าไปในภาพ

การบันทึกผลงานเป็นไฟล์ภาพ

เนื่องจากโปรแกรม Excel มีข้อจำกัดในการบันทึกภาพที่ทำขึ้นมาเป็นไฟล์ จึงต้องมีผู้ช่วยมาช่วยจัดการในเรื่องนี้ โปรแกรม Paint ที่มีแถมมาใน Windows จะช่วยในเรื่องนี้

  1. คลิกที่ภาพใน Excel จากนั้นกด Ctrl + A เพื่อเลือกภาพและตัวหนังสือทั้งหมด
  2. ทำการคัดลอกโดยการกด Ctrl + C
  3. เปิดโปรแกรม Paint ขึ้นมา
  4. วางภาพด้วยการกด Ctrl + V หรือคลิกที่เมนู Home → Paste
  5. หากมีขอบขาว ให้คลิกที่มุมจากนั้นคลิกค้างแล้วเลื่อนจนพอดีภาพ
  6. ทำการบันทึกภาพด้วยเมนู File → Save As
ขั้นตอนการคัดลอกภาพไปยัง Paint เพื่อบันทึกเป็นไฟล์

การตกแต่งภาพขั้นสูง(กว่าปกตินิดนึง)

หากต้องการแต่งภาพและตัวหนังสือมากกว่านี้ โปรแกรม Excel ก็สามารถทำได้ ปรับสี ปรับแสง เปลี่ยนรูปแบบศิลป์ของตัวอักษร ทำได้มากพอตัว อยากรู้ว่าทำอะไรอย่างไรแนะนำให้ลองเล่นดู แล้วจะรู้ว่าโปรแกรม Excel ทำได้มากกว่าสเปรดชีตครับ
